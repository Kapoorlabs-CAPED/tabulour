Tables¶

Move around the table¶

Arrow keys →, ←, ↑, ↓ with Ctrl (or ⌘ in Mac), Shift modifier work as you expects in most of table data editors.

Additionally, Ctrl + mouse wheel zooms in/out the table. Ctrl + Alt + arrow keys scrolls the table to the desired direction.

Edit cells and headers¶

If a table is editable, you can edit the values of cells and headers. Double-clicking, F2 or typing any characters will create an editor for the value.

During editing, the text will always be validated. Invalid text will be shown in red. For the table cells, you can set any validation rules (see Column-wise Settings). For the table headers, duplicated names are not allowed and considered to be invalid.

Obtain the values¶

Ctrl + I in the console will insert a data reference object DATA[...] at the cursor position. The data reference object is updated in real-time when the table selection is changed. This is the fastest way to obtain the values in the table.

Toolbar¶

Toolbar contains many functions that help you with analyzing the table data.

Note

You can “click” any buttons in the toolbar using the keyboard; push Alt (or ⌥ in Mac) to change focus to the toolbar, and follow the tooltip labels to find the appropriate key combo to get to the button you want (similar to Microsoft Office).

File menu¶

  • … Open a table data as a Table from a file using a file dialog.

  • … Open a table data as a SpreadSheet from a file using a file dialog.

  • … Save the currently active table data using a file dialog.

  • … Open a sample data from seaborn.

Table menu¶

  • … Make a copy of the active table as a Table.

  • … Make a copy of the active table as a SpreadSheet.

  • … Call pd.groupby() on the active table.

  • … Call pd.concat() on the active table.

  • … Call pd.pivot() on the active table.

  • … Call pd.melt() on the active table.

  • … Open the finder widget. Several item matching mode (match by text, match by value, partial match and regular expression) are available.

  • … Sort table by a column.

Analyze menu¶

  • … Summarize table data by mean, standard deviation etc.

  • … Evaluate a string expression on the table data. Essentially equivalent to call pd.eval().

  • … Filter table data by a string expression.

  • … Toggle the console widget visibility.

View menu¶

  • … Popup current active table.

  • … Activate dual view mode (horizontal).

  • … Activate dual view mode (vertical).

  • … Reset view mode.

Plot menu¶

  • … Plot table data by plt.plot().

  • … Plot table data by plt.scatter().

  • … Plot histogram of the data by plt.hist().

  • … Create a new figure on the side area.

The embedded plot canvas is interactive. You can also double click the objects in plot canvas to edit its color, line width, etc.

Warning

The matplotlib editor is WIP now. Its behavior may change in the future.
