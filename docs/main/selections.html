Data Type of Table Selections¶

You’ll get table selections using selections property as a SelectionRanges object.

table.selections

SelectionRanges([0:100, 0:1], [2:3, 2:3])

A SelectionRanges object is a list like object except for being immutable. Each selection is a tuple[slice, slice] object.

table.selections[0]

(slice(0, 100, None), slice(0, 1, None))

Thus, you can use them directly in iloc method of DataFrame.

table.data.iloc[table.selections[0]]

More simply, you can use data attribute to get values of selected table.

table.selections.values[0]

    sepal_length
0            5.1
1            4.9
2            4.7
3            4.6
4            5.0
..           ...
95           5.7
96           5.7
97           6.2
98           5.1
99           5.7

[100 rows x 1 columns]

@table.events.selections.connect
def _on_change(sel):
    print(sel)
