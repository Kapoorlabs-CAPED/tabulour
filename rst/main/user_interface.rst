Tables
======

Arrow keys |rarr| |larr| |uarr| |darr| with ``Ctrl``, ``Shift`` modifier work as you
expects in most of table data editors.

.. image:: ../fig/table_interface_0.gif

If a table is editable, you can edit the value from the keyboard. If you are editing
on a ``Table``, data type will be checked.

.. image:: ../fig/table_interface_1.gif


Toolbar
=======

Toolbar contains many functions that help you with the table data.
